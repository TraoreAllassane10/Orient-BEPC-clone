# Description

Orient'Bepc Clone est la reproduction d'une application existante en ligne : [Orient'Bepc](https://orient-bepc.vercel.app/). Cette application permet au élève de 3ème de calculer leurs moyennes de d'orientation après le BEPC.

# Technologies utilisées

- React
- TypeScript
- Tailwind CSS

# Installation

1. Cloner le dépôt

```bash
git clone https://github.com/TraoreAllassane10/Orient-BEPC-clone.git
```

2. Acceder au répectoire

```bash
   cd Orient-BEPC-clone
```

3. Installer les dépendances
```bash
npm install`
```

# Démarrage de l'application

1. Démarrage
```bash
npm run dev
```

2. Ouvre ton navigateur et va sur `http://localhost:5173/` pour commencer à utiliser l'application
